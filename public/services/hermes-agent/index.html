Implementation timeline

From call to live agent — here's the plan.

Most standard deployments are live within 2–3 weeks. Professional builds with complex integrations take 3–4 weeks. Here's exactly what happens at each stage.

Day 1–2

Discovery & scoping call

We learn your business inside and out. What tools you use, what tasks eat your time, which workflows are broken. We map your current stack, identify the highest-impact automations, and spec the exact hardware and agent configuration you need. You leave this call with a written scope document and a fixed price — no surprises.

Day 3–7

Hardware procurement & build

We source every component, assemble the workstation, stress-test it for 48 hours, install Ubuntu Server + Docker, and configure the networking. For BYO equipment clients, we run a remote hardware assessment and OS preparation during this phase instead. Your machine ships (or is ready for remote access) by end of week one.

Day 8–14

Agent installation & customization

Hermes Agent goes in. We install via Docker (sandboxed, secure), configure your LLM providers with automatic failover (cloud primary + local Ollama backup), set up your messaging gateways, and build your custom agent workflows. Every cron job, every skill, every integration — configured and tested against your real data.

Day 14–18

Testing, training & go-live

We run your agent through real-world scenarios — actual client inquiries, real scheduling requests, live data. You and your team get a full training session: how to talk to the agent, how to add tasks, how to check on it. We hand over the operational runbook and flip the switch. Your agent is live.

Day 18–48

Post-launch monitoring & tuning

We monitor performance, review agent logs, tune prompts, and add skills as needed. Your agent gets smarter in real-time as it learns your business. Turnkey clients get 30 days of active support; install-only clients get 14 days. After that, optional Managed Operations keeps us in the loop indefinitely.

Before you deploy.

Hermes Agent is an open-source autonomous AI agent built by NousResearch. Think of it as a digital employee that can read emails, browse the web, execute code, manage files, schedule tasks, and communicate via Telegram, Discord, Slack, or SMS — all running on hardware you own. It has persistent memory (it remembers everything about your business), a skills system (it can learn new capabilities), and supports both cloud AI models (Claude, GPT-4) and local models via Ollama for fully offline operation.

Zero. That's the whole point of hiring us. We handle all the technical setup — hardware, software, networking, security, integrations. Once it's running, you interact with your agent by messaging it on Telegram, Slack, or whatever platform you choose. It's like texting a really capable assistant. If you can send a text message, you can use Hermes.

We configure every deployment with a dual-model setup: a cloud LLM (like Claude or GPT-4) as the primary, and a local model running via Ollama as the fallback. If your internet drops, Hermes automatically switches to the local model. It's slower and less capable, but it keeps running. Your cron jobs, local file processing, and any tasks that don't require external APIs continue uninterrupted.

Running on your own hardware is a massive advantage for compliance. Patient data never leaves your premises when using local models. For cloud LLM calls, we configure the agent to strip PHI before sending prompts. We can also set up a fully air-gapped deployment that uses only local models — no cloud calls at all. That said, we're not compliance attorneys. We build the technical infrastructure to support HIPAA compliance, and we recommend consulting your compliance officer for the final sign-off.
